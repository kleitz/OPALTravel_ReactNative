
## Style guide

### Grays

+ dark `#434343`
+ medium `#95918F`
+ light `#E8E7E6`

### Hues

+ green `#66BB6A`
+ orange `#FCB316`
+ pink `#EC407A`
+ purple `#775CA7`
+ blue `#56C0EE`
+ chartreuse `#CDDC37`

### Typography

+ primary, display, headline `Futura Medium`
+ utility, rubric `Gotham Medium`
    * rubric is always uppercase
+ body `Gotham Narrow Book`
